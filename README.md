<h1>Active Directory Home Lab</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
Built an Active Directory home lab to simulate a corporate windows enivorment. Documented the domain services/sites, automated user and group creation using PowerShell, created/linked GPO's to OU, password reset, results of joining domains between windows 10 & 2019, and dashboard of server manager roles.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
- <b>Windows 2019</b>
