<h1>Active Directory Home Lab</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
Learn to build a functional Active Directory lab using VirtualBox. This tutorial demonstrates setting up a domain controller, configuring network adapters, and installing essential services like DHCP. A PowerShell script automatically creates numerous user accounts.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
